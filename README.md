# 🏥 Claim Approval Prediction (Machine Learning Project)

## Project Overview
This project predicts whether a healthcare insurance claim will be approved or denied using Machine Learning.

## Objective
Build a classification model to predict **ClaimStatus (Approved / Denied)** based on patient and provider data.

---

## Techniques Used
- Data Cleaning & Preprocessing
- Feature Encoding
- Train-Test Split
- Logistic Regression
- Decision Tree Classifier
- Model Evaluation (Accuracy, Confusion Matrix, Classification Report)

---

## Results
- Logistic Regression Accuracy: ~49%
- Decision Tree Accuracy: ~50%

 Model performance is close to random, indicating:
- Weak feature relationships
- Need for better feature engineering
